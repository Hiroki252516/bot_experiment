# Quadratic Equations

A quadratic equation has the form ax^2 + bx + c = 0 where a is not zero.

There are multiple ways to solve a quadratic equation:

1. Factoring, when the expression can be decomposed into linear factors.
2. Completing the square, which transforms the equation into a perfect square form.
3. Using the quadratic formula:

x = (-b ± sqrt(b^2 - 4ac)) / 2a

When teaching beginners, it often helps to:

- start from a concrete numeric example,
- explain each transformation step,
- warn about common sign mistakes,
- check the final answer by substitution.
